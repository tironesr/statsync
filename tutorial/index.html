Step 1: The R Package

Prepare your models for seamless export.

Capture Everything

Use sync_stats() to extract coefficients, confidence intervals, and p-values in one go.

# In R Console
model <- lm(mpg ~ wt, data = mtcars)
sync_stats(model, "Linear Model")

APA-Ready Tables

Format entire regression tables with a single command. Ready for publication standards.

sync_regression_table(
  "Full Model" = model1,
  "Base Model" = model0
)

Export & Ship

Save your results to a .statsync file that Word can read directly.

sync_export(
  model1, model2, 
  file = "results.json"
)

Step 2: The Word Add-in

One-click insertion of your high-precision data.

01

Open the Sidebar

Launch the StatSync taskpane in Microsoft Word. Your R results are automatically grouped by model name, making them easy to browse.

  • Search by coefficient name

Choose Your Format

Select multiple fields for a single coefficient. Insert just the p-value, or a full APA-style report: "β = .45 [0.32, 0.58], p < .001".

Real-Time Sync

Data changed in R? No problem.

The Live Connection

Run sync_serve() in R to start a local server. The Word add-in connects instantly, providing real-time updates as you refine your analysis.

sync_serve(project_name = "Thesis")
# After re-running models:
sync_update()
